Android App Package Download

Click the button below to download the deployment package for building the Android App Bundle (AAB):

Download Deployment Package (2.6MB)

Build Instructions:

  1. Extract the downloaded file:
    tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android folder:
    cd export_package/android
  3. Make gradlew executable (on Mac/Linux):
    chmod +x gradlew
  4. Build the AAB file:
    ./gradlew bundleRelease (Mac/Linux)
    gradlew.bat bundleRelease (Windows)
  5. Find the AAB file at:
    app/build/outputs/bundle/release/app-release.aab

Upload the AAB file to Google Play Console when the build is complete.
