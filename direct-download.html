StrangerWave Deployment Package

This page provides direct access to the StrangerWave deployment package containing everything you need to build the Android App Bundle (AAB) file for Google Play Store submission.

Download Package (2.6MB)

Build Instructions

  1. Extract the package: tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android directory: cd export_package/android
  3. Make the Gradle wrapper executable (Mac/Linux only): chmod +x gradlew
  4. Build the AAB file:
    • On Mac/Linux: ./gradlew bundleRelease
    • On Windows: gradlew.bat bundleRelease
  5. Find the AAB file at: app/build/outputs/bundle/release/app-release.aab
Note: Building an AAB file requires a computer with at least 4GB of RAM and 10GB of free disk space. The build process will not work in the Replit environment due to resource constraints.
