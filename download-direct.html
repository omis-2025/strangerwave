StrangerWave Android App Bundle Builder

Download this package to build the Android App Bundle (AAB) for Google Play Store submission

Deployment Package

This package contains all the necessary files to build an Android App Bundle (AAB) for the StrangerWave app. The AAB format is required for Google Play Store submissions.

Tip: Make sure to build this on a machine with at least 4GB of RAM and sufficient disk space (10GB recommended).
Download Deployment Package (2.5 MB)

Building the AAB Step-by-Step

1

Extract the Package

Open a terminal and run:

tar -xzf strangerwave-deployment-package.tar.gz
2

Navigate to Android Directory

Change to the Android project directory:

cd export_package/android
3

Make Gradle Executable

On Mac/Linux, make the Gradle wrapper executable:

chmod +x gradlew
4

Build the App Bundle

Run the Gradle task to build the bundle:

./gradlew bundleRelease

On Windows, use:

gradlew.bat bundleRelease
5

Locate the AAB File

After a successful build, find your AAB at:

app/build/outputs/bundle/release/app-release.aab
6

Upload to Google Play

Go to Google Play Console and upload the AAB file to your app's production track.

Troubleshooting

Memory Issues

If you encounter memory errors, increase Gradle memory by adding this to gradle.properties:

org.gradle.jvmargs=-Xmx2048m

Keystore Information

The keystore is already configured in the project. If needed for reference:

  • Keystore path: ../../keystores/strangerwave.keystore
  • Keystore password: strangerwave123
  • Key alias: strangerwave
  • Key password: strangerwave123
