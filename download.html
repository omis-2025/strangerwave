Build Instructions

  1. Extract the downloaded file:
    tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android folder:
    cd export_package/android
  3. Make the gradlew file executable (Mac/Linux only):
    chmod +x gradlew
  4. Build the AAB file:
    ./gradlew bundleRelease (Mac/Linux)
    gradlew.bat bundleRelease (Windows)
  5. Find the AAB file at:
    app/build/outputs/bundle/release/app-release.aab
  6. Upload this AAB file to the Google Play Console

System Requirements

  • Java JDK 11 or higher
  • Android SDK (included in the package)
  • At least 4GB of RAM
  • At least 10GB of free disk space
