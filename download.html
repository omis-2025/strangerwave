StrangerWave Deployment Package

The deployment package contains all the necessary files to build the Android App Bundle (AAB) for submission to the Google Play Store.

Package Contents

  • Key configuration files (key.properties)
  • Documentation and supporting files

Next Steps After Downloading

  1. Extract the package on your computer:
    • Windows: Use 7-Zip, WinRAR, or similar tools
    • Mac/Linux: Use tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android project directory:
    • cd export_package/android
  3. Build the AAB:
    • On Mac/Linux: ./gradlew bundleRelease
    • On Windows: gradlew.bat bundleRelease
  4. Find the AAB at:
    • app/build/outputs/bundle/release/app-release.aab
  5. Upload this file to Google Play Console
