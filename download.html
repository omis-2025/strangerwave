AAB Build Instructions

  1. Extract the package on your local machine:
    • Windows: Use 7-Zip, WinRAR, or similar tools
    • Mac/Linux: Use tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android project directory:
    • cd export_package/android
  3. Ensure you have the Android development environment set up:
    • Java JDK 11 or higher installed
    • Android SDK installed (minimum API level 33)
    • JAVA_HOME environment variable properly set
    • At least 4GB of RAM and 10GB of free disk space
  4. Make Gradle executable (Mac/Linux only):
    • chmod +x gradlew
  5. Build the AAB:
    • On Mac/Linux: ./gradlew bundleRelease
    • On Windows: gradlew.bat bundleRelease
    • This process may take several minutes to complete
  6. Find the AAB at:
    • app/build/outputs/bundle/release/app-release.aab
  7. Upload this file to Google Play Console

Troubleshooting Build Issues

  • Memory Issues: If you encounter memory errors, try increasing the Gradle memory allocation by adding this line to gradle.properties: org.gradle.jvmargs=-Xmx2048m
  • Build Errors: Make sure you have the correct Android SDK version installed. The project requires API level 33 or higher.
  • Keystore Issues: The keystore is already included in the package and configured in build.gradle. The keystore password is strangerwave123 and the key alias is strangerwave.
  • Permission Denied: If you encounter permission issues with gradlew, make sure to run chmod +x gradlew to make it executable.
