StrangerWave Deployment Package

Click the button below to download:

Download Package (2.6 MB)

After downloading:

  1. Extract the package: 
    tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to the Android project: 
    cd export_package/android
  3. Build the AAB: 
    ./gradlew bundleRelease
    On Windows use: 
    gradlew.bat bundleRelease
  4. Find the AAB file at: 
    app/build/outputs/bundle/release/app-release.aab
  5. Upload this file to Google Play Console
