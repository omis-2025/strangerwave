Build Instructions:

  1. Extract the file: tar -xzf strangerwave-deployment-package.tar.gz
  2. Navigate to Android folder: cd export_package/android
  3. Make gradlew executable: chmod +x gradlew (Mac/Linux only)
  4. Build the AAB: ./gradlew bundleRelease (or gradlew.bat bundleRelease on Windows)
  5. Find the AAB file at: app/build/outputs/bundle/release/app-release.aab

Upload the AAB file to Google Play Console when the build is complete.
